## How to Deploy on GitHub Pages
1. Push all files to your GitHub repository (e.g., `DLR-Solutions`).
2. Go to your repository settings on GitHub.
3. Find the "Pages" section.
4. Set the source branch to `main` (or `master`) and the folder to `/root` (or `/docs` if you move files).
5. Save and visit the provided URL.
